# Pipes-Puzzule

<H2> 게임 소개 및 개요</H2>
램덤으로 생성되는 파이프들을 연결해서 제한된 사용 횟수 내에서 파이프를 연결하는 게임입니다.
 <H3>개발 중점 요소</H3>
<ul>
 <li>BFS, DFS 알고리즘을 활용하여 파이프 퍼즐을 동적 생성.</li>
 <li>게임 요소별 객체화</li>
</ul>
